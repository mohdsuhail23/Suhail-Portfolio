# **App Name**: DevSphere Portfolio

## Core Features:

- Dynamic Project Showcase: Fetch, display, and manage project details (title, thumbnail, summary, tech stack, links) from Sanity CMS with dynamic routes and SEO-optimized individual pages.
- Experience Timeline: Render a chronological vertical timeline of professional experiences, including company, role, duration, and key bullet points, fetched from Sanity CMS.
- Hero Section & Navigation: Implement a prominent hero section with headline, value statement, and CTA buttons. Includes a sticky, blurring navbar with smooth-scroll navigation and a mobile hamburger menu.
- Contact Section: Provide direct links for email, LinkedIn, and GitHub, offering clear call-to-actions for communication.
- SEO & Performance Optimization: Ensure high Lighthouse scores, fast Core Web Vitals, and optimal search engine indexing through SSG, Next.js Metadata API, dynamic OG tags, structured data, sitemaps, and robots.txt.
- Sanity CMS Integration: Seamless integration with Sanity.io for content management, using defined schemas for Projects and Experience, GROQ queries, and Portable Text rendering.
- Image Optimization: Utilize 'next/image' and Sanity Image Builder for responsive, WebP-converted, lazy-loaded images with proper alt tags to ensure high performance.

## Style Guidelines:

- The palette is dark and professional, reflecting the requested Zinc/Slate theme. Primary: #8C4DF0 (a vibrant deep indigo/purple) to signify creativity and modernity against the dark background. Background: #201B24 (a deep, desaturated violet-grey) providing a solid foundation. Accent: #727EFD (a brighter, slightly desaturated blue-purple) for highlights and interactive elements.
- Headlines use 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel. Body text uses 'Inter' (sans-serif) for its readability and neutral, objective aesthetic, ensuring clear communication of detailed project and experience information.
- All icons will be sourced from Lucide React, ensuring consistency with the clean, professional, and developer-focused aesthetic.
- Responsive layout design that prioritizes a minimalist and professional appearance. Key elements include a sticky navbar with a blur background, a vertical experience timeline, and a grid-based project display with clean card UI and large hero images on detail pages.
- Subtle animations are incorporated to enhance user experience without being distracting. This includes smooth scroll for navigation, hover effects on project cards, and potential use of Framer Motion for tasteful transitions.
